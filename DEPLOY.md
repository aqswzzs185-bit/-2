# 👵 부모님 생활 블로그 자동화 도우미 - 24시간 무료 클라우드 배포 가이드

본 가이드는 **내 컴퓨터를 꺼두어도 휴대폰만으로 언제 어디서나 접속하여 프로그램을 사용**할 수 있도록 구글/네이버 같은 대형 클라우드 서버에 프로그램을 24시간 켜두는 방법을 설명합니다.

모든 서비스는 **100% 무료**로 이용하실 수 있습니다.

---

## 🚀 24시간 배포를 위한 5단계 (딱 한 번만 해두면 평생 무료!)

### 1단계: 깃허브(GitHub) 계정 만들고 코드 올리기
1. [GitHub 공식 사이트](https://github.com/)에 회원가입을 합니다.
2. 가입 후 화면 우측 상단의 **[New]** 또는 **[Create repository]** 단추를 눌러 개인 코드 보관함을 만듭니다. (이름은 자유롭게 지정, 예: `parents-blog-bot`)
3. 내 컴퓨터에 있는 프로그램 폴더 전체 파일들을 드래그 앤 드롭하여 이 보관함에 업로드(Upload)해 둡니다.

### 2단계: Streamlit Cloud 무료 가입
1. [Streamlit Community Cloud](https://streamlit.io/cloud)에 들어갑니다.
2. **[Sign up]**을 누르고, 방금 1단계에서 만든 **[GitHub 계정으로 로그인]**을 선택해 연동 가입합니다.

### 3단계: 내 프로그램을 클라우드 서버에 켜기 (Deploy)
1. 로그인 후 대시보드 화면 우측 상단의 **[New app]** (새 앱 만들기) 단추를 누릅니다.
2. 아래 빈칸을 기입합니다:
   - **Repository**: 1단계에서 만든 내 코드 보관함 선택
   - **Main file path**: `app.py` 입력
3. 맨 아래 파란색 **[Deploy!]** 단추를 누릅니다.

### 4단계: 3분간 커피 한 잔 대기
- 클라우드 컴퓨터가 스스로 내 프로그램의 파이썬 환경을 세팅하고 가동을 시작합니다. 
- 잠시 기다리면 화면에 나만의 영구적인 **24시간 전용 인터넷 주소**가 발급되며 사이트가 켜집니다!
- (예: `https://parents-blog-bot.streamlit.app`)

### 5단계: 휴대폰에 나만의 24시간 주소 등록 및 사용
1. 휴대폰 인터넷 창으로 방금 발급된 나만의 전용 인터넷 주소에 접속합니다.
2. 휴대폰 인터넷 브라우저 설정에서 **[홈 화면에 추가]** 또는 **[앱 설치]**를 누릅니다.
3. 이제 **컴퓨터가 꺼져 있어도** 휴대폰 바탕화면에 생성된 아이콘만 누르면 24시간 언제 어디서나 포스팅 초안 생성, 제목 재생성, 이미지 프롬프트 복사가 작동합니다!

---

## 💡 클라우드 모드로 휴대폰에서 글 올리는 꿀팁!

컴퓨터가 꺼져 있을 때는 클라우드 컴퓨터가 내 네이버 로그인 창을 자동으로 타이핑해줄 수 없습니다. 대신 다음처럼 모바일 복사 붙여넣기로 10초 만에 글을 올리실 수 있습니다.

1. 휴대폰 앱에서 포스팅 기획 후 `글 초안 생성` 단추를 누릅니다.
2. 검수와 수정이 끝나면 `블로그 입력`을 누릅니다.
3. 화면에 나타나는 **[🟢 클라우드 모드 전용 복사 포스팅 패널]**에서:
   - **[제목 복사]** 아이콘 터치 ➡️ 내 휴대폰의 **네이버 블로그 공식 앱**을 열고 제목란에 붙여넣기!
   - **[본문 복사]** 아이콘 터치 ➡️ 본문란에 붙여넣기!
   - **[해시태그 복사]** 아이콘 터치 ➡️ 태그란에 붙여넣기!
4. 등록 단추를 누르면 끝! 

내 PC의 크롬 프로필 세션을 클라우드에 노출하지 않으므로 **네이버 해킹이나 보안 유출 우려가 0%인 가장 안전한 24시간 모바일 포스팅 방법**입니다.
