# 👵 부모님 생활 블로그 자동화 도우미 (Parents Life Blog Automation Assistant)

본 프로그램은 **부모님 선물, 시니어 생활용품, 혼자 사는 부모님 안심용품** 등을 메인 주제로 네이버 블로그 포스팅을 기획 및 자동 생성하고, 안전하게 네이버 블로그 스마트에디터 ONE에 **임시저장**할 수 있도록 돕는 일체형 통합 자동화 도구입니다.

사용자가 메인 키워드, 상품군, 쇼커 링크 정보만 입력해도 자녀의 진심이 담긴 정보성 포스팅을 손쉽게 완성할 수 있습니다.

---

## 🌟 18대 핵심 기능

1. **키워드 입력**: 대시보드를 통해 메인 타겟 키워드와 상품군을 간편하게 기입합니다.
2. **글 유형 자동 판단**: 선택한 글 유형(선물 추천형, 생활문제 해결형 등)에 맞춰 상품 DB 지식 기반의 맞춤식 전개를 지원합니다.
3. **제목 후보 5개 생성**: 6대 기획 유형(문제해결/선물추천/체크리스트/계절/비교/공감형)에 부합하는 다채로운 제목 후보를 지어냅니다.
4. **최종 제목 추천**: AI가 제안한 제목 중 가독성과 클릭율이 가장 우수할 1종을 추천 뱃지와 함께 제안합니다.
5. **네이버 블로그용 본문 생성**: 10단계 블로그 작성 원칙에 따라 자녀의 따뜻한 시선과 시니어 건강 팩트가 조화된 고품질 원고를 생성합니다.
6. **쇼커 링크 자연 삽입**: 도입부 기재 배제, 문맥 유도 텍스트 삽입 등 링크 삽입 10가지 규칙에 맞추어 본문 내 쇼커 링크를 1~2회만 삽입합니다.
7. **광고성 표현 검수**: "무조건", "대박", "필수 구매" 등의 과장 단어를 탐지하여 경고 및 대체제를 권장합니다.
8. **의학적 단정 표현 검수**: "완치", "치료 효과 완벽" 등 의료법에 저촉될 위험 소지가 있는 단정을 감지 및 회피합니다.
9. **SEO 검수**: C-Rank 검색 엔진과 가독성 지표를 실시간 평정하고, 결함 발생 시 **🔧 Auto-Fix** 단추로 자동 교정합니다.
10. **이미지 프롬프트 생성**: 동아시아/한국 시니어 10대 명세(텍스트/로고/워터마크 배제)를 적용한 3종 실사 이미지 영문 프롬프트를 자동 생성합니다.
11. **해시태그 생성**: 네이버 블로그 검색 트렌드를 고려한 연관 태그 10개를 쉼표 구분 형태로 제공합니다.
12. **미리보기**: 모바일 앱 화면 레이아웃 규격에 맞춰 원고의 줄바꿈과 가독성을 즉석 시뮬레이션할 수 있습니다.
13. **네이버 블로그 자동 입력**: Playwright 세션을 기동하여 임시저장 에디터의 제목, 본문(하이퍼링크 서식 활성화 주입), 해시태그를 자동 타이핑합니다.
14. **임시저장**: 에디터 우측 상단의 저장 단추를 자동 클릭하여 네이버 서버에 저장하고, 브라우저 락(Lock)을 소거하여 리소스를 회수합니다.
15. **글 저장 및 재사용**: 과거 작성 이력을 안전하게 영구 저장소(`posts_history.json`)에 보관하여 다시 불러오거나 편집할 수 있습니다.
16. **중복 글 방지**: 키워드/제품군 7일 내 작성 대조 및 유사 제목 60%/도입부/링크구문 자가복제 검사 및 4대 다변화 각도 우회 제안 UI
17. **오류 로그 기록**: 실행 중 오류가 발생할 경우, 브라우저 창을 닫지 않고 원인을 파악해 컨트롤 타워 하단에 상세 로그를 출력합니다.
18. **README 작성**: 바로 보고 계시는 본 사용자 통합 가이드라인 문서입니다.

## 👵 10대 중요 작성 및 자동화 원칙

1. **정보성 위주 전개**: 노골적인 홍보가 아닌, 부모님 생활 속 문제를 짚는 진정성 있는 꿀팁처럼 읽혀야 합니다.
2. **자녀의 따뜻한 공감**: 부모님 안부를 챙기는 다정하고 다소 염려스러운 가족의 관점을 유지합니다.
3. **링크 2회 이하 제한**: 흐름을 해치지 않고 정보로 느껴지도록 쇼커 링크는 절대 2회를 넘지 않습니다.
4. **과장 표현 일체 금지**: "최고", "필수 구매", "대박" 같은 구매 압박 단어는 쓰지 않습니다.
5. **의학적 효능 단정 금지**: "통증 완화에 도움을 줍니다" 처럼 완화나 예방 수치로 표기하며 의학적 효능을 확답하지 않습니다.
6. **불안 자극 회피**: 고령 부모님의 무력함이나 불안을 지나치게 자극하지 않고 희망적인 안전 대책을 전합니다.
7. **자동 발행 배제**: 실수를 예방하기 위해 최종 포스팅 발행은 오직 사용자가 원고 검토 후 수동으로 진행합니다.
8. **임시저장까지만 자동화**: 프로그램은 스마트에디터 내 임시 저장까지만 안전하게 마칩니다.
9. **로그인 정보 안전 관리**: 사용자의 네이버 아이디/비밀번호는 일절 하드디스크나 파일에 저장하지 않고 쿠키 세션 프로필만 로컬에 유지합니다.
10. **자가복제 원천 예방**: 매일 사용해도 네이버로부터 유사문서 분류를 당하지 않도록 제목 어조, 도입부, 해시태그 구성을 다채롭게 바꿉니다.

## 🛠️ 설치 방법 (초보자용 3분 가이드)

1. **파이썬(Python) 설치**:
   - 컴퓨터에 파이썬이 설치되어 있지 않다면 [Python 공식 홈페이지](https://www.python.org/downloads/)에서 최신 버전을 다운로드하여 설치합니다.
   - *주의*: 설치 시 반드시 **Add Python.exe to PATH** 체크박스에 체크해 주세요.

2. **의존 패키지 설치**:
   - 명령 프롬프트(cmd) 또는 터미널을 열고 본 폴더 경로로 이동하여 아래 명령어를 순서대로 입력합니다:
     ```bash
     pip install -r requirements.txt
     playwright install chromium
     ```

## 🚀 실행 및 모바일 앱 접속 방법 (PC / 휴대폰 멀티 플랫폼)

본 프로그램은 모바일 반응형 웹 기술이 적용되어 PC뿐만 아니라 **휴대폰(스마트폰)에서도 전용 앱처럼 접속하여 자유롭게 포스팅을 기획하고 제어**할 수 있습니다.

### 방법 1: PC에서 단독 실행 시
- 본 폴더 내의 **[run.bat](file:///c:/Users/admin/Desktop/AI%20자동화%20프로그램%20개발/네이버%20블로그%20글쓰기%20자동화/run.bat)** 파일을 더블클릭하여 실행합니다.

### 방법 2: 스마트폰/외부 기기에서 앱 형태로 원격 실행 시 (추천)
1. 본 폴더 내의 **[run_mobile.bat](file:///c:/Users/admin/Desktop/AI%20자동화%20프로그램%20개발/네이버%20블로그%20글쓰기%20자동화/run_mobile.bat)** 파일을 더블클릭하여 실행합니다.
2. **동일 공유기(Wi-Fi) 망인 경우**:
   - PC의 로컬 IP 주소(예: `192.168.0.x`)를 확인한 뒤, 휴대폰 브라우저 주소창에 `http://192.168.0.x:8501`을 입력해 접속합니다.
3. **외부망(5G/LTE/타사 와이파이)인 경우**:
   - [ngrok 공식 사이트](https://ngrok.com/)에서 무료 가입 후 authtoken을 PC에 등록해 두면, `run_mobile.bat` 실행 시 전 세계 어디서든 스마트폰으로 접속할 수 있는 보안 도메인 주소(`https://xxxx.ngrok-free.app`)가 생성되어 언제 어디서나 외부에서 PC 제어가 가능해집니다.
4. **📱 전용 모바일 앱 형태로 추가하기 (PWA 팁)**:
   - 스마트폰 브라우저(사파리, 크롬, 삼성인터넷 등)로 위의 접속 주소에 들어갑니다.
   - 브라우저 메뉴에서 **[홈 화면에 추가]** 또는 **[앱 설치]** 단추를 누릅니다.
   - 스마트폰 바탕화면에 **"부모님 생활 블로그 자동화 도우미"** 앱 아이콘이 생성되어, 터치 한 번으로 언제 어디서든 PC 자동화대를 조작하는 네이티브 앱처럼 기동됩니다!

## 🕹️ 권장 사용 순서 (컨트롤 타워 네비게이터)

마스터 컨트롤 타워의 버튼 순서에 맞춰 진행하면 가장 안전하고 빠르게 고품질 글을 임시저장할 수 있습니다:

1. **환경 설정 (최초 1회)**:
   - 왼쪽 사이드바에 본인의 `Google Gemini API Key`와 `Naver ID`를 입력하고 **[설정 저장]**을 누릅니다.
2. **[🆕 1. 새 글 만들기]**:
   - 기존의 작업대를 깨끗이 청소하고 새로운 포스팅 집필 상태를 엽니다.
3. **[🔑 2. 키워드 입력]**:
   - 왼쪽 입력란에서 메인 키워드(예: `70대 부모님 발마사지기`), 소개할 상품군 및 쇼커 링크들을 지정합니다.
4. **[✍️ 3. 글 초안 생성]**:
   - AI가 글 작성을 시작합니다. 만약 과거 작성 이력과 겹친다면 **중복 경고 패널**이 뜨며 어조를 다변화할 4가지 대안 각도를 제안하므로, 마음에 드는 각도를 선택하여 우회 생성하세요.
5. **[🔄 4. 제목 다시 생성]**:
   - 본문은 그대로 둔 채, 우측 화면 상단에 6대 기획 유형(문제해결, 공감 등) 뱃지가 붙은 제목 후보 5종 중 하나를 클릭해 최종 제목을 자유롭게 선택합니다.
6. **[🔗 5. 쇼커 링크 삽입]**:
   - 본문 중간과 마무리에 쇼커 링크 안내 문구를 물리 규칙에 맞춰 최종 조립합니다.
7. **[🔍 6. 광고성 검수]** & **[📊 7. SEO 검수]**:
   - 법적 위반 표현이 있는지 확인하고, 검색 노출 점수가 낮다면 **[Auto-Fix]** 버튼을 눌러 교정합니다.
8. **[🎨 8. 이미지 프롬프트]**:
   - 글 내용에 꼭 맞는 따뜻한 시니어 홈 라이프스타일 사진을 그리기 위한 영문 프롬프트를 확인하고 복사하여 이미지 AI에 사용합니다.
9. **[👁️ 9. 미리보기]**:
   - 줄바꿈과 문단 너비가 모바일 화면에서 자연스럽게 읽히는지 스크롤하며 검토합니다.
10. **[📤 10. 블로그 입력]**:
    - Playwright 브라우저 창이 열리며 네이버 블로그 에디터에 원고 정보가 타이핑됩니다.
    - *참고*: 최초 실행 시 네이버 로그인 화면이 뜨면 직접 1회만 로그인해주시면, 다음부터는 로그인 없이 다이렉트로 기입이 수행됩니다.
11. **[💾 11. 임시저장]**:
    - 자동 작성 완료 후 이 버튼을 클릭하면, 열려 있던 블로그 페이지의 **[저장]**(임시저장) 단추가 자동으로 클릭 완료되고 브라우저가 안전하게 수거 회수됩니다.
12. **[📂 12. 저장된 글 목록]**:
    - 하단 이력 관리 탭을 통해 작성했던 글을 언제든 편집기로 다시 로드해 재입력할 수 있습니다.
